# دليل الملفات - Velora Theme
## File Guide - Understanding Each File

---

## 📁 المجلدات والملفات

### views/ - ملفات القوالب (Twig Templates)

#### 📄 views/layouts/master.twig
**الغرض:** القالب الأساسي الرئيسي
**يحتوي على:**
- بنية HTML الأساسية
- استيراد الخطوط والأيقونات
- Tailwind CSS و Font Awesome
- رأس الموقع والتذييل
- متغيرات CSS للألوان

**متى يتم تعديله:**
- تغيير خطوط الموقع
- إضافة مكتبات جديدة (CDN)
- تغيير الألوان الأساسية
- تعديل بنية HTML الأساسية

---

#### 📄 views/pages/index.twig
**الغرض:** صفحة الرئيسية
**يحتوي على:**
- Hero Section (قسم البطل)
- Features Grid (شبكة المميزات)
- Products Grid (شبكة المنتجات)
- Testimonials (التقييمات)
- Newsletter Signup (اشتراك النشرة الإخبارية)

**متى يتم تعديله:**
- تغيير نصوص الصفحة الرئيسية
- إضافة أقسام جديدة
- تعديل الصور والأيقونات
- تغيير CTA buttons

---

#### 📄 views/pages/product.twig
**الغرض:** صفحة تفاصيل المنتج الفردي
**يحتوي على:**
- Breadcrumb Navigation (التنقل الهرمي)
- Product Image Gallery (معرض الصور)
- Product Information (اسم، سعر، وصف)
- Add to Cart Button (زر إضافة للسلة)
- Product Features (مميزات المنتج)
- Related Products (منتجات ذات صلة)

**متى يتم تعديله:**
- تغيير تخطيط صفحة المنتج
- إضافة أقسام إضافية
- تعديل عدد المنتجات ذات الصلة
- تغيير ترتيب العناصر

---

#### 📄 views/components/header.twig
**الغرض:** مكون رأس الموقع (قابل لإعادة الاستخدام)
**يحتوي على:**
- شريط علوي (Country, Phone, Login)
- شريط التنقل الرئيسي
- شعار الموقع
- روابط التنقل الأساسية
- أيقونات البحث والمفضلة والسلة
- قائمة محمولة

**المميزات:**
- Sticky Navigation (ثابت عند التمرير)
- Search Overlay (بحث متقدم)
- Mobile Menu (قائمة محمولة قابلة للطي)

**متى يتم تعديله:**
- تغيير الشعار
- إضافة/حذف روابط التنقل
- تغيير ألوان الشريط
- تعديل رقم الهاتف أو البلد

---

#### 📄 views/components/footer.twig
**الغرض:** مكون تذييل الموقع
**يحتوي على:**
- معلومات العلامة التجارية
- روابط التصنيفات
- روابط مهمة (خصوصية، شروط)
- معلومات الاتصال
- أيقونات وسائل التواصل
- طرق الدفع
- إحصائيات
- حقوق النشر
- زر العودة للأعلى

**متى يتم تعديله:**
- تغيير معلومات الاتصال
- إضافة روابط اجتماعية جديدة
- تحديث طرق الدفع
- تغيير حقوق النشر

---

#### 📄 views/components/product-card.twig
**الغرض:** بطاقة المنتج (قابلة لإعادة الاستخدام)
**يحتوي على:**
- صورة المنتج
- شارة الفئة
- زر المفضلة
- اسم المنتج
- الوصف المختصر
- السعر
- زر إضافة للسلة

**المميزات:**
- Hover Effects (تأثيرات عند الحركة)
- Wishlist Toggle (إضافة/حذف من المفضلة)
- Add to Cart Event (حدث إضافة للسلة)

**متى يتم تعديله:**
- تغيير تخطيط البطاقة
- إضافة عناصر جديدة
- تعديل التأثيرات عند الحركة

---

### assets/ - الملفات الثابتة

#### 📄 assets/css/app.css
**الغرض:** ملف CSS الرئيسي
**يحتوي على:**
- CSS Custom Properties (متغيرات الألوان)
- Global Styling (أنماط عامة)
- Custom Animations (حركات مخصصة)
- Typography Styles (أنماط الخط)
- Component Styles (أنماط المكونات)
- Responsive Breakpoints (نقاط التوقف)
- Accessibility Features (ميزات الوصول)

**الحركات المتضمنة:**
- fadeInUp / fadeInDown
- shimmer / float
- pulse-glow / pulse

**متى يتم تعديله:**
- تغيير الألوان
- إضافة حركات جديدة
- تعديل الأنماط العامة
- تحسين الأداء

---

#### 📄 assets/js/app.js
**الغرض:** ملف JavaScript الرئيسي
**يحتوي على:**
- Mobile Menu Handler
- Search Toggle & Overlay
- Cart Management
- Wishlist Functionality
- Image Gallery
- Intersection Observer (حركات التمرير)
- Newsletter Form
- Notifications System

**الوظائف الرئيسية:**
1. معالج الشريط اللزج (Sticky Navbar)
2. معالج القائمة المحمولة
3. معالج البحث المتقدم
4. إدارة السلة والمفضلة
5. الحركات والتأثيرات
6. التنبيهات والإشعارات

**متى يتم تعديله:**
- إضافة وظائف تفاعلية جديدة
- تحسين الأداء
- إصلاح الأخطاء
- تكامل مع APIs جديدة

---

### locales/ - ملفات الترجمة

#### 📄 locales/ar.json
**الغرض:** الترجمات العربية
**يحتوي على:**
- نصوص الرأس والقائمة
- نصوص البطل والمميزات
- نصوص المنتجات والتقييمات
- رسائل الإخطار والأخطاء
- نصوص التذييل

**البنية:**
```json
{
  "header": { ... },
  "products": { ... },
  "errors": { ... }
}
```

**متى يتم تعديله:**
- تغيير أي نص في الموقع
- إضافة ترجمات جديدة
- تحديث الرسائل

---

#### 📄 locales/en.json
**الغرض:** الترجمات الإنجليزية
**نفس البنية مثل ar.json**

### 📄 tailwind.config.js
**الغرض:** إعدادات Tailwind CSS
**يحتوي على:**
- ألوان مخصصة (gold, dark)
- خطوط مخصصة (Tajawal)
- حركات مخصصة
- أحجام مخصصة
- متغيرات متقدمة

**متى يتم تعديله:**
- تغيير نظام الألوان
- إضافة خطوط جديدة
- إضافة حركات جديدة
- تخصيص المتغيرات

**الألوان المتاحة:**
- gold-light, gold, gold-dark
- dark-bg, dark-card, dark-border

### 📄 package.json
**الغرض:** إعدادات المشروع والاعتماديات
**يحتوي على:**
- معلومات المشروع
- الأوامر المتاحة (scripts)
- الاعتماديات (dependencies)
- معلومات Salla

**الأوامر:**
```bash
npm run dev     # تطوير مع المراقبة
npm run build   # بناء للإنتاج
npm run lint    # التحقق من الكود
npm run test    # تشغيل الاختبارات
```

## 🗂️ خريطة العلاقات

```
master.twig (البنية الأساسية)
    ├─ header.twig (رأس الموقع)
    ├─ [محتوى الصفحة]
    ├─ footer.twig (التذييل)
    ├─ app.js (وظائف الموقع)
    └─ app.css (تنسيقات الموقع)

index.twig (الصفحة الرئيسية)
    ├─ يستخدم product-card.twig
    └─ يستخدم master.twig

product.twig (صفحة المنتج)
    ├─ يستخدم product-card.twig (منتجات ذات صلة)
    └─ يستخدم master.twig

product-card.twig (بطاقة المنتج)
    └─ تُستخدم في index.twig و product.twig

locales/ (الترجمات)
    ├─ ar.json (عربي)
    └─ en.json (إنجليزي)

assets/ (الملفات الثابتة)
    ├─ css/app.css
    └─ js/app.js

config (الإعدادات)
    ├─ tailwind.config.js
    └─ package.json
```

## 📊 جدول التعديلات الشائعة

| ماذا تريد؟ | أي ملف؟ | ماذا تفعل؟ |
|----------|--------|---------|
| تغيير الألوان | tailwind.config.js | عدّل في القسم colors |
| تغيير النصوص | locales/ar.json | عدّل النصوص العربية |
| تعديل الرئيسية | views/pages/index.twig | عدّل المحتوى |
| إضافة رابط | views/components/header.twig | أضف `<a>` جديد |
| تغيير الشعار | views/components/header.twig | غيّر النص والصورة |
| تغيير أنماط | assets/css/app.css | أضف CSS جديد |
| إضافة وظيفة | assets/js/app.js | أضف JavaScript |
| معلومات الاتصال | views/components/footer.twig | عدّل البيانات |

## 🔍 كيفية البحث عن شيء ما

**تريد تغيير جملة معينة؟**
1. افتح المحرر وابحث (Ctrl+Shift+F)
2. ابحث عن الجملة
3. سيخبرك أين هي موجودة

**تريد إضافة صفحة جديدة؟**
1. انسخ views/pages/index.twig
2. سمّها باسمك (مثل about.twig)
3. عدّل المحتوى

**تريد تغيير رنك الموقع؟**
1. ادهب إلى tailwind.config.js
2. غيّر أكواد الألوان
3. جميع الأماكن ستتحدث تلقائياً
