================================================================================
🚀 البدء السريع
================================================================================

1. تثبيت الاعتماديات:
   $ npm install

2. التطوير مع المراقبة:
   $ npm run dev

3. بناء الإصدار الإنتاجي:
   $ npm run build

4. فتح المشروع في محرر (مثل VS Code):
   $ code .

🎨 نظام الألوان
================================================================================

الثيم يستخدم نظام ألوان احترافي:

الذهب (Gold):
  - Gold Light:   #E8D4A8  (ذهب فاتح)
  - Gold Default: #C9A84C  (ذهب أساسي)
  - Gold Dark:    #A07D2D  (ذهب غامق)

الخلفية الداكنة (Dark):
  - Dark Background: #0A0A0F (خلفية داكنة جداً)
  - Dark Card:       #12121A (لون البطاقات)
  - Dark Border:     #1F1F2E (لون الحدود)

يمكنك تغيير هذه الألوان في: tailwind.config.js

🛠️ التخصيص
================================================================================

التعديلات الشائعة سهلة جداً:

تغيير الألوان:
  → tailwind.config.js (البحث عن colors)

تغيير النصوص:
  → locales/ar.json (البحث عن النص)

تغيير الشعار:
  → views/components/header.twig (البحث عن VELORA)

إضافة رابط جديد:
  → views/components/header.twig (إضافة <a> جديد)

تعديل الصفحة الرئيسية:
  → views/pages/index.twig (تعديل المحتوى)

  {# حلقة عرض المنتجات #}

  {% endfor %}

❓ الأسئلة الشائعة
================================================================================

س: كيف أبدأ الآن؟
ج: 1) npm install
   2) npm run dev

س: هل الثيم متجاوب؟
